# 스타트업(Startup) 코드란?

프로그램이 작동되기 위해서 필요한 준비 과정을 위한 코드로 여러 초기화 작업과 C에서 사용할 메모리(스택, 힙) 등을 할당하고 마지막에 main() 함수를 호출하여 프로그램을 동작하게 한다.

3. 초기화되지 않은 데이터 영역 0으로 초기화
4. 스택을 위한 공간을 할당하고 초기화
